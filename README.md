# Videogames
Site jeux vidéos TP

zoning (Zyed) :
  barre latérale (zone de gauche) : naviguer entre les pages
  centre (zone du milieu) : page
  en-tête (zone du haut) : logo à gauche, nom et barre de recherche au milieu, boutons "connexion" et "s'inscrire" (ou "compte" à droite)
  pied-de-page (zone du bas) : copyrights, contacts, cookies, charte d'utilisation, mentions légales, "qui sommes-nous"

page d'accueil (Zyed, Gad) :
  sondage (indisponible sans connexion, 1 seule fois max, obligatoire)
  articles d'actualités jeux (preview)

page d'articles (Gad) :
  articles d'actualité jeux
  commentaires

page de connexion/création de compte (Alexis) :
  username
  mot de passe
  Captcha de vérification

page de compte (Alexis) :
  photo de profil
  username
  nom visible

page admin (Alexis) :
  utilisateurs (nom, photo de profil)

modération automatique (Zyed, Alexis) :
  photo de profil (nudité, ect... vérification)
  nom/commentaires (dictionnaire de mots et leurs variables interdits)
